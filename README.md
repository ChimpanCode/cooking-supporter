# 献立決めをサポートするアプリケーション

晩御飯に何を作るか決めるのがめんどくさい時のための意思決定 Web アプリケーション
(開発中)

## 使用方法

1. 作りたい/作ったことのあるレシピを登録（料理名, 画像, 元 Recipe の URL, 検索に用いる材料）
2. サイドバーから使用したい材料を選択
3. 検索結果からレシピを閲覧
